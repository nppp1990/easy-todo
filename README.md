# easy-todo

新人练手项目：仿MAC自带应用《**提醒事项**》

> 我写的纯单机应用！虽然是模仿、但是这个自带应用的功能和细节太多了，大概就完成了不到30%吧

以下是动图演示（网络问题、得等一等才能看到）
![show](https://raw.github.com/nppp1990/easy-todo/master/doc/img/gif1.gif)

大家可以按照下面的需求文档从0到1，实现一个简单的todo网页应用，以此来学习vue3
